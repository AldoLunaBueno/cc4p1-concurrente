# Tarea sockets

Tomé los archivos de redestalk y solo modifiqué los archivos Cliente50.java y Servidor50.java, los cuales pasan a llamarse TareaCliente.java y TareaServidor.java. El cliente ahora al conectarse al servidor a través del socket puede recibir como mensaje una cadena como "sumar 1 2" e interpretarla como un comando: realizar la suma 1 + 2 = 3, tras lo cual retorna el resultado como mensaje de respuesta al servidor.
